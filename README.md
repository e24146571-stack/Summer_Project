# RPG 自律管理系統

## Project 簡介
把每天的自律行為(如讀書、健身、課外學習等等)轉換成 RPG 角色屬性經驗值得個人管理系統。

## 屬性系統
1. **INT** (智力): 本科課業(工數、電子學、電路學、計組、機統)
2. **STR** (身體健康): 健身房、長跑訓練
3. **SAN** (精神值): 情緒調適、壓力紓解
4. **WIS** (智慧): 課外自我拓展(學習英文、日文、專案開發)
5. **CHA** (魅力): 社交、領導

## Tech Stack
1. Python 3.14
2. Streamlit
3. SQLite + SQLAlchemy

## Project 狀態
開發中

## 開發環境設置
1. 建立虛擬環境: 'python -m venv venv'
2. 啟動虛擬環境: 'venv\Scripts\activate' (Windows)
3. 安裝套件: 'pip install -r requirements.txt'
